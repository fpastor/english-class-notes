# English Class Notes

This repository contains class notes and materials for English classes. The structure of the repository is organized as follows:

```
campus.live/
  ├─ assets/
  ├─ class-notes/
  ├─ level-2/
  ├─ level-3/
  └─ online-class-pdf/
  ```

- `assets/`: Contains images, icons, and other media files used in the class notes.
- `class-notes/`: Contains the main class notes for various topics covered in the English classes.
- `level-2/`: Contains materials and notes specifically for level 2 English classes.
- `level-3/`: Contains materials and notes specifically for level 3 English classes.
- `online-class-pdf/`: Contains PDF versions of the class notes for online classes.
